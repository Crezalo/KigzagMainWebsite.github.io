How does NFT Stake work?

1. Creators generate their digital artwork.



2. Creator mint NFT out of his artwork and add it to his creator vault. Creator can also add already minted NFTs.



3. Vault issues ERC20 creator tokens against the NFT. 118888 tokens are issued when a given creator's vault is created. Vault doesn't allow minting of new tokens hence as and when new NFTs are added the price of individual token should go up in a long run by design.



4. By default 76% of Creator Tokens are minted to vault address and rest 24% to creator addess. Creator can start the Initial Creator Token Offering anytime after the vault is created with specified minimum price and ERC20 base token like WETH, USDC, USDT, etc. This offering is for specified time.




5. Maximum of 38% of creator tokens are allowed to be bought from the vault under the initial creator token offering. Buyers buy creator tokens directly from the creator's vault contract by paying the base token, at or above the minimum price, to the vault. It is first come first serve basis.



6. After the offering closes, creator tokens from the vault, as much amount as sold in the offering, are used to create liquidity pools along with all the base tokens received. Initially vault owned 76% of creator tokens, suppose 32% of these tokens are sold then to match this 32% another 32% from the vault is sent to the liquidity pool, leaving 12% in vault, and since 32% of creator tokens are worth all the base tokens received hence all base tokens are sent to the liquidity pool to create a trading pair for this creator.



7. Creator tokens left in the vault, 12% in this case, are sent to the creator address



8. Now that creator's liquidity pools are ready, trading between creator token and base token can happen. Since we have as much liquidity in pool (32% in above case) as much creator tokens bought (32% in above case), it is a self-sufficient scenario for trading to begin.



9. If pool contains 500 WETH and 50000 Creator Token, then quote price of creator token is 0.01 WETH currently. Price of last Creator Token left in the pool, in case all creator tokens are swapped against WETH, will be 25000000 WETH and if selling of creator token continues then price of last creator token added will be 0.00000004WETH. Basic Idea of Automated Market Making.





10. Owner of creator token can redeem NFT from vault against proportionate amount of tokens. Suppose there are 100 NFTs in vault and someone owns 1% of all tokens of a creator, he can get one NFT and corresponding tokens will be transfered to the creator.



11. Creato can add new many NFTs to vault. The more the NFTs in vault, the lesser tokens required to redeem an NFT. This also ensures that in longer range of time same number of tokens can buy more NFT hence a pontential price rise for the token in future.



12. Creator gets royalty fee(0.05%) upon each swap from the pool which creator can change as per convenience. Small incentives motivates creator to continue adding more NFTs to vault.



13. Creator can liquidate only 2% of creator token in a month to ensure trust and stability of creator token's lquidity pool.
